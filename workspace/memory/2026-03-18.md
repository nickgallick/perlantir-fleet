## Cron Lesson Learned
- CRITICAL: For autonomous overnight builds, must use `agentTurn` payload with `sessionTarget: "isolated"`, NOT `systemEvent` with `sessionTarget: "main"`. systemEvent just injects text — doesn't trigger the agent to act unless a user message comes in.
