# Nick Brand System

Use this as the default brand consistency skill.

## Focus
- visual consistency
- recurring style language
- typography/color discipline
- stronger recognizable feel

## Hard rules
- consistency beats novelty
- trust and clarity matter more than trendiness
- brand system should support conversion and usability
