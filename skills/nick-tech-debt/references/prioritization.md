# Prioritization

## Use these lenses
- How much does this slow changes?
- How risky is it when touched?
- How often will we touch it?
- How hard is it to fix?
- Does fixing it unlock faster shipping?

## Buckets
### Blocker
- actively blocks releases or safe changes

### Slows shipping
- repeatedly costs time or causes avoidable bugs

### Cleanup later
- real but low urgency
