# Deployment And Memory

## Deployment rules
- deploy after meaningful changes
- verify the live URL
- smoke test the live app

## Memory capture
After major project work, store:
- project name
- path
- live URL
- major tech decisions
- key fixes
- follow-up items

## Next-step behavior
After shipping, suggest logical next steps such as:
- analytics
- waitlist/email capture
- auth hardening
- admin tools
- custom domain
- SEO polish
