# Admin And Ops Defaults

For internal/admin tools, prefer:
- clear dashboards over flashy UI
- strong filtering and search
- audit-friendly tables
- empty states that explain next actions
- role-aware access boundaries
- bulk actions with confirmation when needed

For client-facing SaaS apps, prefer:
- clear onboarding
- obvious primary CTA
- trust-building UI
- short time-to-value
- polished auth and billing-adjacent flows
