# Build Checklist

## Before build
- Define user, goal, and core action
- Define main entities and auth model
- Define MVP scope

## During build
- Happy path works
- Validation works
- Error handling exists
- Empty states exist
- Mobile works
- Basic accessibility works

## Before deploy
- No secret exposure
- Env vars wired correctly
- Core flow tested
- Console errors checked
- Metadata/title sane

## After deploy
- Live URL captured
- QA run completed
- Memory updated with project name, URL, tech decisions, follow-ups
- Next logical improvements suggested
