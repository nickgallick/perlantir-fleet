# UX Research Methodology

## Screen-by-Screen Analysis
1. Map complete user journey (visit → signup → first value → repeat)
2. Document each screen: purpose, primary action, secondary actions
3. Note information hierarchy: what's biggest, what's first
4. Identify friction points (confusion, extra steps)
5. Identify delight points (surprisingly good)

## Conversion Funnel
- Landing → Sign up (what convinces?)
- Sign up → First action (time to value?)
- First action → Repeat (what brings back?)
- Free → Paid (what triggers upgrade?)

## Patterns to Document
- Navigation: tabs, sidebar, bottom bar, hamburger
- Onboarding: steps, progressive disclosure, skip
- Empty states: first-time experience design
- Loading: skeletons, spinners, progressive
- Error handling: inline, toast, modal
- Social proof: reviews, testimonials, stats
- CTAs: placement, color, copy, urgency

## Mobile-Specific
- Thumb zone: primary actions bottom 1/3
- Gestures: swipe, pull-to-refresh, long press
- Bottom sheet vs modal
- Tab bar organization (max 5)
