# Skill: Distribution & Customer Acquisition Analysis

An idea with no path to customers is a hobby project. Map the EXACT route to the first 100 paying customers.

## Step 1: Where Do These Customers Congregate?

For the target persona, find SPECIFIC communities with actual sizes:
- Subreddits (list each with subscriber count)
- Slack/Discord communities (search "[industry] slack community" or "[industry] discord")
- Twitter/X accounts and hashtags they follow
- Newsletters they read (search "[industry] newsletter" — check subscriber counts on Substack, Beehiiv)
- Facebook groups (still massive for real estate, healthcare, teachers, local businesses)
- LinkedIn groups (for B2B/professional personas)
- Conferences and events (for positioning, not day-1 sales)
- Forums (niche industry forums still exist and have loyal users)
- YouTube channels covering this space (potential partnership or ad targets)
- Podcasts in this niche (potential sponsorship or guest appearance)

## Step 2: Rate Distribution Channels (1-5 for this specific idea)

- Product Hunt launch: best for dev tools, SaaS, consumer apps with visual appeal
- Reddit/HN organic: best for technical audiences, niche communities (Nick can post authentically)
- SEO/Content marketing: best long-term but slow. Rate higher if search volume exists for the problem.
- TikTok / short-form video: best for visual products, consumer, creator tools. RATE HIGHER if overlaps with @golfscenarios or @ogfinancebro audiences.
- Cold email outreach: best for high-ticket B2B ($200+/mo). Bad for everything else.
- Partnerships / integrations: best for tools that plug into existing workflows (Shopify app, Slack app, Zapier integration)
- Freemium / viral: best for collaboration tools, anything with network effects
- Marketplace listing: best for Chrome extensions, WordPress plugins, Shopify apps, Slack apps (built-in distribution)
- Community seeding: best when a clear, active community exists — become a helpful member, then introduce the tool

## Step 3: Nick's Distribution Edge Check

Does this idea overlap with assets Nick already has?
- @golfscenarios audience (golf, sports, entertainment) → if yes, free distribution to 13K+ engaged followers
- @ogfinancebro audience (finance, money, business) → if yes, free distribution
- Perlantir / AI builder community presence → if yes, credibility in dev tools space
- Professional network in financial services → if yes, warm intros for fintech products
- If NONE of these apply, distribution is harder. Factor that in.

## Step 4: Timeline to 100 Paying Customers

- < 1 month: STRONG distribution fit. Active community, product-led growth, existing audience overlap.
- 1-3 months: DECENT channels. Requires content creation, community engagement, or outreach effort.
- 3-6 months: HARDER distribution. Need to build audience from scratch or do sustained outbound.
- 6+ months: Enterprise sales cycle or no clear community. RED FLAG for Perlantir.

## Distribution Difficulty Rating
- EASY: Clear community exists, Nick has audience overlap, product-led growth possible
- MEDIUM: Communities exist but Nick needs to earn credibility there, moderate content effort needed
- HARD: No obvious community, need to build audience from scratch, or requires sales-heavy approach
