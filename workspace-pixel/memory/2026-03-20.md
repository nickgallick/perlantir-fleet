## V0 Pipeline Learnings
- V0 MCP `sendChatMessage` can timeout on complex fixes — keep fix messages SHORT and focused
- Demo URLs update with each V0 iteration (new URL per message)
- Headless Playwright screenshots may not trigger scroll events — always verify sections render without scroll
- Reference site analysis (Death Wish Coffee + Coffee Collective) was valuable for establishing design direction before generation
- Image-first approach for photography-forward designs: generate hero → generate section images → send all URLs in single V0 prompt
