# MathMind Project — In Progress (2026-03-21)

## Project Overview
Premium K-12 math app (ages 3-18). 13 screens total × 4 age-adaptive design tiers = up to 52 variants. React Native / Expo.

## Tiers
1. **Playroom** (Pre-K, K): Warm cream (#FFF8F0), coral (#FF8C6B), soft, rounded-2xl, 80pt touch targets, audio-first, max 3-4 interactive elements
2. **Explorer** (Grades 1-4): White-gray (#F8F9FA), blue (#5B8DEF), 56pt touch targets, balanced
3. **Academy** (Grades 5-8): Light/dark available (#F4F5F7 / #1A1B2E), purple (#6C5CE7), analytics-forward
4. **Studio** (Grades 9-12): White/dark (#FFFFFF / #0D1117), blue (#3B82F6), dark default, zero cartoons, looks like Linear/Notion

## Design System Notes
- All screens mobile-first (393×852, iPhone 15 Pro)
- Nunito font for youngest tiers, Inter for middle, SF Pro for oldest
- Touch targets: Playroom 80pt, Explorer 56pt, Academy 48pt, Studio 44pt
- NO whileInView animations (causes invisible sections in V0 demo)
- Generous whitespace always
- Math rendering: proper fractions, exponents, square roots (use react-native-math-view equivalent)
- Haptic feedback on every answer submission
- Sound design: subtle, not arcade

## Key Decisions
- Generate backward from clean answers for all math problems
- Target 85% success rate per student
- Elo-based adaptive difficulty (not flat leveling)
- Spaced repetition for mastered skills (1, 3, 7, 14, 30, 60, 120 day intervals)
- NO IXL-style punishment — encouragement always
- COPPA compliant: parental consent, no third-party analytics, parental gate on purchases/links
