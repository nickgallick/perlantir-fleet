# Growth Engine Design

Design at least three loops per product.

## For each loop define
- Trigger action
- Distribution mechanism
- Conversion action
- Cycle time
- Estimated viral coefficient (K)
- Dependencies and failure points

## Core Rule
Prefer loops where normal product use naturally creates distribution.

## Required Outputs
- North Star Metric
- Loop inventory table
- Highest-K loop with reason
- Friction reduction ideas
- Instrumentation events needed to measure loop health
