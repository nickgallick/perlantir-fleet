# Link Building for Developer Products

## Strategy: Earn Links Through Value, Not Outreach

Developer audiences are link-resistant to outreach. The best links come from creating things worth linking to.

## Tier 1: Linkable Assets (Build These)

### Open Source Components
- Open source your SDK/connector → GitHub stars + README links back to product
- Contribute to related open source projects → contributor profile links
- Create useful utilities related to your domain (ELO calculator, model classifier)

### Data & Research
- Publish unique data from your platform: "State of AI Agent Performance 2026"
- Weekly/monthly data reports that ML researchers and bloggers cite
- Interactive data visualizations (live leaderboard embed, model comparison charts)

### Documentation & Guides
- Best-in-class docs that people link to as references
- "How to build X" tutorials that rank and earn natural links
- Comparison pages (Arena vs Kaggle vs HumanEval — honest, detailed)

## Tier 2: Community-Driven Links

### Developer Directories & Lists
- Submit to awesome-* lists on GitHub (awesome-ai-tools, awesome-machine-learning)
- Product directories: AlternativeTo, Product Hunt, DevHunt, SaaSHub
- AI tool directories: There's an AI for That, Futurepedia, AI Tool Directory

### Conference & Meetup Presence
- Speak at AI/ML meetups → talk pages link to your product
- Sponsor hackathons → event pages link to sponsors
- Submit to conference CFPs → accepted talks get linked from program pages

### Community Content
- Guest posts on Dev.to, Hashnode, Medium (with canonical)
- Podcast appearances (show notes link to your product)
- YouTube collaborations with AI content creators

## Tier 3: Outreach (Use Sparingly)

### Broken Link Reclamation
1. Find pages that link to dead AI benchmark/competition sites
2. Contact the page owner: "Hey, your link to [dead site] is broken. We built [Arena] which covers similar ground."
3. Low conversion rate but high-quality links when it works.

### Unlinked Brand Mentions
1. Search for "Agent Arena" mentions without links
2. Contact: "Thanks for mentioning us! Would you mind adding a link?"
3. Highest conversion rate of any outreach tactic.

### Resource Page Inclusion
1. Find "best AI tools" or "AI resources" pages
2. Submit your product with a brief pitch
3. Works best when your product is genuinely useful to their audience

## Link Targets (Arena-Specific)

### Pages to Build Links To:
1. Homepage (brand authority)
2. /leaderboard (data-driven, update frequently — fresh content earns more links)
3. /blog/[pillar-pages] (educational content — most natural link targets)
4. /docs (developer reference — earns links from tutorials and Stack Overflow)

### Link Quality Criteria:
- Domain Rating 30+ (anything below isn't worth the effort)
- Topically relevant (AI, dev tools, coding, benchmarks)
- Real traffic (not a link farm)
- Dofollow (nofollow links have minimal SEO value)

## Monthly Link Building Cadence

| Activity | Frequency | Expected Links |
|----------|-----------|----------------|
| Publish data report | Monthly | 3-5 organic links |
| Awesome-list submissions | One-time + updates | 2-3 permanent links |
| Directory submissions | One-time | 5-10 permanent links |
| Guest posts / Dev.to | 2x/month | 2-4 links/month |
| Broken link outreach | Weekly (1h) | 1-2 links/month |
| Community engagement | Daily | 1-2 organic links/week |

## Anti-Patterns (Will Hurt You)
- Buying links from link farms
- Mass automated outreach
- Comment spam with links
- PBN (Private Blog Network) links
- Reciprocal link schemes ("I'll link you if you link me")
- Low-quality directory submissions (100+ directories = spam signal)
