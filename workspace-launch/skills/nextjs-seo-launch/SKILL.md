# Next.js SEO Launch

Complete SEO setup for Next.js apps deployed on Vercel. Covers pre-launch technical checklist and ongoing content SEO strategy.

## Pre-Launch Checklist → references/pre-launch-checklist.md
Everything that must be in place before the first real user sees the site. Meta tags, OG images, structured data, sitemap, robots.txt, canonical URLs.

## Next.js-Specific SEO → references/nextjs-patterns.md
SSR vs ISR vs static — how each affects indexing. Dynamic meta tags with generateMetadata. App Router SEO patterns. Common Next.js SEO mistakes.

## Core Web Vitals → references/core-web-vitals.md
LCP, INP, CLS targets and how to hit them on Next.js + Vercel. Font loading, image optimization, layout shift prevention.

## Content SEO Strategy → references/content-strategy.md
Topic clusters, keyword targeting, internal linking, blog setup for long-tail traffic.

## Link Building → references/link-building.md
Earning backlinks for developer products. Digital PR, content assets, resource page outreach.

## Workflow

1. Read references/pre-launch-checklist.md — implement everything before launch
2. Read references/nextjs-patterns.md — verify Next.js-specific patterns are correct
3. Read references/core-web-vitals.md — test and optimize performance
4. Post-launch: read references/content-strategy.md for ongoing SEO
5. Month 2+: read references/link-building.md for authority building
