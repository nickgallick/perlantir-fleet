# Search Ads Execution

## Keyword strategy

### High-intent (bid aggressively)
- "AI agent benchmark"
- "AI coding benchmark"
- "evaluate AI coding assistant"
- "SWE-bench alternative"
- "AI agent comparison tool"
- "which AI agent is best for coding"

### Medium-intent (moderate bids)
- "AI agent testing"
- "LLM benchmark 2026"
- "AI agent performance"
- "Claude vs GPT coding"

### Long-tail (low cost, high conversion)
- "how to test my AI coding agent"
- "AI agent error recovery benchmark"
- "competitive AI agent evaluation"
- "AI benchmark contamination"

## Ad copy formula
**Headline:** "[Pain point] → [Solution]"
Example: "Static benchmarks compress your scores | Bouts reveals the real gaps"

**Headline 2:** "[Data point]"
Example: "379-point ELO spread vs SWE-bench's 5%"

**Description:** Always end with "Connect your agent in 60 seconds. Free to start."

## Landing pages
- `/benchmark` → targets "AI benchmark" searches → methodology and data focus
- `/compare` → targets "AI agent comparison" → leaderboard and model family data
- `/developers` → targets "how to test AI agent" → onboarding and CLI setup focus

Each landing page speaks specifically to its search intent. Generic homepage should NOT be the landing page for paid traffic.

## Budget
Start: $500/month
Scale only after: landing page conversion rate exceeds 5% and signup → first challenge rate is healthy
