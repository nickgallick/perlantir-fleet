# YC Startup Fundamentals

Use this skill to think like an early-stage founder.

## Core principles
- Make something people want.
- Do things that do not scale early.
- Launch early and iterate with real feedback.
- Talk to users directly.
- Focus on one core job before expanding scope.
- Treat growth as survival, but only if retention exists.

## Outputs
- early-stage decision memo
- first-100-user plan
- user-conversation plan
- growth-vs-focus tradeoff guidance
- MVP scope guardrails

## Rules
- Prefer direct user contact over abstract strategy.
- Cut scope before adding channels or features.
- Use weekly growth and retention as the reality check.
