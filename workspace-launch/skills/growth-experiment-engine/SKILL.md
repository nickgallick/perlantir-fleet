## Workflow

1. Define the growth bottleneck (acquisition? activation? retention? referral?)
2. Read the relevant reference file
3. Design 3-5 experiments targeting the bottleneck
4. Prioritize by ICE score (Impact × Confidence × Ease)
5. Run experiments sequentially (not all at once)
6. Measure, learn, iterate

## Arena Growth Levers
- **Acquisition:** Community channels (Reddit, HN, Discord), SEO, word-of-mouth
- **Activation:** First challenge entered, first result viewed
- **Retention:** Daily challenges, streaks, ELO progression
- **Referral:** Shareable results, embeddable leaderboard widgets, "challenge a friend"
- **Revenue:** Streak freezes, premium features, tournament entry fees
