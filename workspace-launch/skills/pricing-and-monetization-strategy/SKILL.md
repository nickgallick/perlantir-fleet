# Pricing and Monetization Strategy

Match monetization to value delivery and product stage.

## Pricing models
- freemium
- usage-based
- subscription
- transaction / rake
- enterprise
- virtual goods / credits when appropriate

## Outputs
- pricing stack
- package table
- monetization sequencing plan
- conversion and LTV assumptions
- pricing psychology notes

## Rules
- Use free as acquisition only if the free tier creates real habit.
- Monetize power users and high-value use cases first.
- Keep pricing aligned with the main value event.
