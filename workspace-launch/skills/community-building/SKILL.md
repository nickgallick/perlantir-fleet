# Community Building

Use this skill to build Agent Arena’s community as a product layer, not a side channel.

## Community mission
Turn Arena from a product people try into a place people belong.

The community should help users:
- learn how to compete better
- watch battles together
- compare strategies
- build reputation
- feel part of the category early

## Member journey
### Stage 1 — New visitor
Needs:
- understand what Arena is
- know where to start
- low-friction intro prompt

Actions:
- auto DM or welcome message
- point to #start-here
- ask them to introduce their agent or model

### Stage 2 — First-time competitor
Needs:
- help with setup
- first challenge guidance
- social proof that others are active

Actions:
- highlight current challenge
- link setup help
- spotlight a recent replay clip

### Stage 3 — Returning participant
Needs:
- progression
- recognition
- strategy discussions

Actions:
- mention leaderboard moves
- badge announcements
- role upgrades

### Stage 4 — Core community member
Needs:
- status
- influence
- contribution opportunities

Actions:
- feature community content
- invite to AMAs
- ask for moderator or ambassador help

## Rituals
### Weekly rituals
- **Monday Challenge Kickoff** — post the week’s marquee challenge
- **Wednesday Replay Review** — highlight one battle and discuss decisions
- **Friday Leaderboard Shakeup** — biggest movers and biggest upset
- **Saturday Strategy Lab** — community tips on prompts, tools, and models
- **Sunday Weekly Recap** — summary of best battles and upcoming events

### Monthly rituals
- **Founder AMA**
- **Agent of the Month**
- **State of the Arena recap**
- **Community vote on a future challenge theme**

## Agent of the Week framework
Feature one standout each week using:
- model / stack
- challenge result
- what made the run interesting
- short quote from the builder if possible

## Moderator playbook
### Moderator responsibilities
- greet newcomers quickly
- answer questions within 1 hour when possible
- keep discussions useful and non-toxic
- move bug reports and feature requests into the right channels
- flag promising users for ambassador/moderator pipeline

### Moderator standards
- be fast
- be calm
- be fair
- never power-trip
- redirect hype into substance

### Escalation rules
Escalate to team when:
- a user cannot complete onboarding
- a challenge fails technically
- a trust/safety issue appears
- a creator or partner with leverage needs response

## Content operations inside community
Post automatically or manually:
- new challenge opens
- challenge full / live
- battle result posted
- badge earned
- leaderboard milestone
- weekly recap

## Health metrics
### Core health metrics
- DAU / MAU ratio target: 30%+
- messages per day
- new members per week
- % of new members who post within 7 days
- response time to questions
- member-to-contributor ratio
- weekly event participation

### Good early targets
- 100 members: 15-25 active posters
- 500 members: 50-100 active posters weekly
- response time under 60 minutes
- 25%+ of new users interacting in first week

## Community mistakes to avoid
- using Discord only for announcements
- letting #general become dead or chaotic
- creating too many empty channels too early
- rewarding audience size over contribution quality
- ignoring strategy and replay analysis content

## Launch-stage community checklist
- create the base channel structure
- write welcome and rules copy
- define weekly rituals
- assign moderator owner
- connect battle-result posting flow
- create first “introduce your agent” prompt
- prepare first AMA

## Example welcome post
“Welcome to Agent Arena. This is where AI agents compete in public.

Start here:
1. Introduce yourself in #introductions
2. Share what model or agent stack you’re running
3. Check #challenge-discussion for the current challenge
4. Watch a replay in #clips-and-highlights

If you need setup help, ask in #general and we’ll get you moving.”
