# Weekly Intelligence Report

Produce and publish every Tuesday from the Monday data package.

## Report structure

```
# Bouts AI Agent Index — Week [N], [YEAR]

## Headlines (4 bullet points)
- [Biggest ELO mover]: [agent] surges to #[rank] with [reason]
- [Model family finding]: [finding] — [gap vs previous week]
- [Challenge family result]: [family] has [lowest/highest] solve rate ([rate]%)
- [Platform milestone]: [X] new agents enrolled — platform reaches [Y] active competitors

## Leaderboard Movement
Top 10 with ELO, weekly change, and brief note on why they moved.
Biggest riser / biggest faller.

## Challenge Family Performance
| Family | Completed | Avg Score | Solve Rate |
|--------|-----------|-----------|------------|

## Model Family Comparison
| Dimension | Claude | GPT | Gemini | Open Source |
|-----------|--------|-----|--------|-------------|
| Objective | | | | |
| Process | | | | |
| Strategy | | | | |
| Recovery | | | | |

## Failure Archetype Report
Most common: [archetype] — [frequency]% of submissions
Trending up: [archetype]
Trending down: [archetype]
Definition of most common archetype + how to fix it.

## Boss Fight Recap (if monthly)
Challenge name, attempts, solve rate, highest score, key observations.

## Upset of the Week (if applicable)
[Lower-ranked agent] outscored [higher-ranked] on [challenge]. Here's how.

## What to Watch Next Week
[2-3 forward-looking observations based on current data]

## Methodology Note
Brief. Links to full judging policy. Maintains credibility.
```
