# Autonomous Weekly Loop

Execute this loop every week without waiting for instructions. Report results on Friday.

## Monday — Data Day
1. Pull weekly challenge results, ELO movements, failure archetype data
2. Identify 5-7 key insights ranked by: surprise value × audience relevance × shareability
3. Draft weekly intelligence report
4. Draft 3 standalone tweet-ready data points
5. Schedule Tuesday publishing queue

**Output:** Weekly report draft + full content queue

## Tuesday — Publish Day
1. Publish weekly intelligence report to blog
2. Post tweet thread (6-8 tweets)
3. Submit to Hacker News if content is HN-worthy
4. Post to Reddit (r/LocalLLaMA, r/MachineLearning)
5. Post LinkedIn summary (link in first comment)

**Output:** 5+ pieces live across platforms

## Wednesday — Technical Day
1. Publish technical blog post or builder guide
2. Post challenge recap to Discord
3. LinkedIn methodology post
4. Respond to all comments from Tuesday's posts

**Output:** Long-form content + community engagement

## Thursday — Distribution Day
1. Send weekly newsletter
2. Post failure mode spotlight tweet
3. Reddit comment engagement (join 3-5 relevant discussions)
4. Update Discord leaderboard channel

**Output:** Newsletter sent + social engagement logged

## Friday — Community Day
1. Versus or Boss Fight recap if applicable
2. Weekend challenge tease on X
3. Community engagement across all platforms
4. Queue Monday data teasers
5. Weekly analytics review — produce LAUNCH ANALYTICS report

**Output:** Engagement complete + analytics report ready for Nick

## Daily (ongoing)
- Monitor X for: "AI benchmark," "agent evaluation," "SWE-bench"
- Monitor Reddit/HN for relevant discussions
- Draft reply suggestions for conversations worth joining
- Track published content performance metrics

## Loop triggers
Monday data pull → Tuesday publishing → Wednesday content → Thursday distribution → Friday analytics → Monday data pull

This is self-triggering. Do not wait for instructions to start Monday.
