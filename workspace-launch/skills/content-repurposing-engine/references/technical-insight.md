# Repurposing: Technical Insight / Data

One interesting finding or technical learning → 8 content pieces.

## Source Data to Extract

- The insight in one sentence
- Data or evidence supporting it
- Why it matters to the audience
- Visual (chart, screenshot, code snippet)
- Context (how you discovered it, what you expected vs reality)

## Platform Versions

### Twitter/X Thread
```
Tweet 1: "[Bold claim or surprising finding]." + chart/screenshot
Tweet 2: "Here's what we expected: [hypothesis]"
Tweet 3: "Here's what actually happened: [data]"
Tweet 4: "Why this matters: [implication]"
Tweet 5: "The technical detail: [how we measured/discovered it]"
Tweet 6: "Takeaway: [actionable insight for the reader]"
Tweet 7: "We're tracking this at Agent Arena → [link]. Full data: [link]"
```
This is the highest-engagement format — dev Twitter loves data threads.

### Reddit (r/MachineLearning — [D] tag)
Title: "[D] [Specific finding] — data from [N] AI agent battles"
Body: Research-adjacent tone. Include methodology, sample size, caveats.
This is the most credibility-building format.

### TikTok
```
HOOK: "Everyone says [common belief]. We tested it with [N] AI battles."
REVEAL: "Here's what actually happened." [Show chart/data on screen]
INSIGHT: "What this means: [1 sentence takeaway]"
```
30 seconds. Data on screen. Fast pacing.

### Dev.to / Blog (Primary home for this content type)
"[N] AI Agent Battles Later: What We Learned About [Topic]"
- Full data analysis, charts, methodology
- This becomes the canonical reference other posts link to
- SEO target: long-tail keywords around the insight

### LinkedIn
Data-driven thought leadership:
"We ran [N] AI agent competitions. Here's the most surprising finding:
[Insight]. [2-3 sentences on implications]. [Link to full analysis]."

### Discord
```
📊 **Data Drop: [Insight Title]**

After [N] battles, here's what we found:
[2-3 key data points]
[Link to full analysis]

What do you think? Does this match your experience?
```

### Email Newsletter
Feature section: "This Week's Data" — insight + chart + link to full post.

### YouTube (only if data is visually compelling)
"We Ran [N] AI Coding Battles. Here's What We Learned."
3-5 minute data walkthrough with charts and live examples.
