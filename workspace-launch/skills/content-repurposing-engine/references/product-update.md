# Repurposing: Product Update / Feature Launch

One new feature or milestone → 8 content pieces.

## Source Data to Extract

- Feature name and what it does
- Problem it solves (why users wanted it)
- Screenshot or demo recording of the feature
- Technical detail worth sharing (architecture, stack decision)
- User impact (who benefits, how)

## Platform Versions

### Twitter/X Thread
```
Tweet 1: "Just shipped: [Feature]. Here's why it matters." + screenshot
Tweet 2: The problem — what was broken/missing before
Tweet 3: The solution — what the feature does (with GIF or screenshot)
Tweet 4: Technical angle — how we built it, interesting decision
Tweet 5: What's next — roadmap tease
Tweet 6: Try it now → [link]
```

### Reddit
**Subreddit-specific framing:**
- r/SideProject: "Added [feature] to Agent Arena — here's the build story and what I learned"
- r/webdev: "How I built [technical feature] with [stack] — code walkthrough"
- r/LocalLLaMA: "[Feature] now supports local models — here's how to connect yours"

**Body:** 300-600 words. Lead with the problem, show the solution, include technical detail.

### TikTok Script (30s)
```
HOOK: "I just added [feature] to my AI competition platform."
DEMO: 15s screen recording showing the feature
PAYOFF: "Link in bio if you want to try it."
```

### Discord Announcement
```
🚀 **New Feature: [Name]**

[2-3 sentences: what it does and why]

[Screenshot]

Try it now: [link]
Feedback? Drop it in #feature-requests
```

### Email Newsletter
```
New: [Feature Name]

[1 sentence what it does]
[1 sentence why we built it — user request or problem identified]
[1 sentence how to use it]

[CTA button: Try [Feature]]
```

### Dev.to Post
"How We Built [Feature] — [Technical Angle]"
- Architecture decisions and tradeoffs
- Code snippets for the interesting parts
- Performance considerations
- What we'd do differently

### LinkedIn
Professional angle — what this feature means for AI evaluation, the industry, or Nick's building journey.

### YouTube Short (optional)
Quick 45s demo: problem → feature → result.
