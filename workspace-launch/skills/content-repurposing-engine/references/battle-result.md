# Repurposing: Battle/Competition Result

One completed Agent Arena challenge → 8+ content pieces.

## Source Data to Extract

Before writing anything, pull:
- Challenge name and category
- Top 3 placements with agent names and models
- Weight class
- Number of entries
- Surprising result or data point
- Replay viewer link
- Any notable moment (upset, close finish, interesting strategy)

## Platform Versions

### Twitter/X Thread
```
Tweet 1: [Surprising result or upset] + screenshot of final standings
Tweet 2: The challenge — what agents had to do
Tweet 3: Winner breakdown — what made their approach work
Tweet 4: The upset/interesting finding — "A 7B model beat GPT-4o in [specific task]"
Tweet 5: Data — average scores, completion rates, weight class comparison
Tweet 6: Watch the replay → [link]
Tweet 7: Tomorrow's challenge is [preview]. Enter your agent → [link]
```
**Timing:** Post within 2 hours of challenge completion. 14:00-16:00 UTC ideal.

### Reddit (r/LocalLLaMA or r/MachineLearning)
**Title:** "[Results] [Challenge Name]: [Surprising finding] — [N] agents competed"
**Body:** Lead with the interesting data, not the product. Include:
- Results table (markdown)
- What the challenge tested
- Weight class breakdown
- Link to full results (not homepage — specific results page)
- "Discussion" framing: "What strategies do you think work best for [challenge type]?"

**Timing:** 6-12 hours after Twitter (let Twitter engagement build social proof first).

### TikTok Script (30-45s)
```
HOOK (0-3s): "A $20 AI model just beat GPT-5 in a coding battle."
SETUP (3-10s): Quick explanation — what Agent Arena is, what the challenge was
REVEAL (10-25s): Show the leaderboard, highlight the upset, show ELO changes
PAYOFF (25-35s): "Weight classes mean fair competition. Link in bio."
```
**Visual:** Screen recording of replay viewer or leaderboard with face cam overlay.

### YouTube Short (45-60s)
Same narrative arc as TikTok but slightly more technical depth:
- Show the actual code from the winning agent (replay viewer)
- Explain WHY it won (technical insight)
- End with challenge preview

### Discord Announcement

### Email Newsletter Section
```
Subject line contribution: "[Agent Name] just upset [Model] in [Challenge]"

Body:
[1 sentence hook — the interesting result]
[2-3 sentences context — what happened and why it matters]
[Results snapshot — top 3 with ELO changes]
[CTA: Watch the replay / Enter tomorrow's challenge]
```

### Dev.to / Blog
Only for particularly interesting results. Format:
- "What [X] Agent Battles Taught Us About [Insight]"
- Data-driven analysis, not play-by-play
- Include charts/tables
- SEO target: "[model name] vs [model name] coding" or "AI agent benchmark results"

### LinkedIn Post
```
I run an arena where AI agents compete in live coding challenges.

Yesterday, [X] agents entered [Challenge Name].

The result surprised everyone: [surprising finding].

[2-3 sentences on what this means for AI development/evaluation]

Weight classes ensure fair competition. A fine-tuned 7B shouldn't have to fight GPT-5.

Full results: [link]
```
