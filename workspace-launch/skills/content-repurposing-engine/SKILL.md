# Content Repurposing Engine

Take one content event and produce platform-optimized versions for every distribution channel. No generic "just post it everywhere" — each version is native to the platform's format, voice, and audience expectations.

## Content Source Types

Each source type has a different repurposing playbook:

### Battle/Competition Result → references/battle-result.md
A completed Agent Arena challenge with results, rankings, and interesting data.

### Product Update/Feature Launch → references/product-update.md
New feature, major improvement, or milestone announcement.

### Technical Insight/Data → references/technical-insight.md
Something learned from building, running, or analyzing the product.

### User Story/Community Highlight → references/community-highlight.md
Notable user achievement, interesting agent strategy, or community moment.

## Workflow

1. Identify the source content type
2. Read the corresponding reference file
3. Extract the core story, data points, and visual assets
4. Produce all platform versions in one pass
5. Include posting schedule (stagger across 48-72 hours)

## Platform Output Checklist

Every repurposing pass produces:

| Platform | Format | Length | Visual |
|----------|--------|--------|--------|
| Twitter/X | Thread (5-10 tweets) | ~1,400 chars total | Screenshot or GIF |
| Reddit | Text post | 300-800 words | Inline images |
| TikTok | Script + shot list | 30-60s | Screen recording |
| YouTube Short | Script | 30-60s | Vertical screen capture |
| Discord | Announcement | 100-200 words | Embed image |
| Email newsletter | Section | 150-300 words | Header image |
| Dev.to / blog | Full post or update | 500-1500 words | Screenshots + code |
| LinkedIn | Post | 200-400 words | Image or carousel |

## Rules

- Never copy-paste the same text across platforms
- Each version must feel native to the platform
- Stagger posts: Twitter first (momentum), Reddit 6-12h later, others across 48-72h
- Every version needs a different hook — same story, different entry point
- Include specific posting time recommendations per platform
- Visual assets must be sized for each platform (Twitter 16:9, TikTok 9:16, etc.)
