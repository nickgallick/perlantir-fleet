# App Store Keyword Research

## Keyword Selection Framework

### Step 1: Seed Keywords
Start with 3 categories:
- **Brand terms**: App name, company name, product name
- **Category terms**: What category the app fits (e.g., "coding", "AI", "competition")
- **Problem terms**: What users search when they have the problem you solve

### Step 2: Expansion
For each seed keyword, find:
- Synonyms and variations
- Long-tail phrases (3-5 words)
- Competitor brand names you can rank near
- Misspellings users commonly make

### Step 3: Scoring
Score each keyword on:
- **Search volume**: How many people search this (tools: Sensor Tower, AppTweak, data.ai)
- **Relevance**: How closely it matches your app (1-10 scale)
- **Competition**: How many apps target this keyword (low/medium/high)
- **Conversion potential**: Will searchers actually want your app?

### Step 4: Prioritization Matrix

| Priority | Volume | Competition | Relevance |
|----------|--------|-------------|-----------|
| P1 (must have) | Any | Low-Medium | 9-10 |
| P2 (target) | High | Medium | 7-8 |
| P3 (nice to have) | High | High | 6-7 |
| Skip | Low | High | Any |

### iOS Keyword Field Strategy (100 chars)
- No spaces after commas (wastes characters)
- No need to repeat words in title/subtitle
- Singular form only (Apple matches plural automatically)
- No prepositions (the, a, an, for, with)
- Test: can you read the keyword list and understand what the app does?

**Arena example:**
```
ai,agent,compete,arena,elo,rating,leaderboard,code,challenge,battle,rank,model,benchmark,live
```

### Google Play Description Keywords
Google Play indexes the full description. Repeat target keywords 3-5x naturally.
- Primary keyword in first sentence
- Secondary keywords in feature bullets
- Long-tail phrases in use case descriptions
- Don't keyword stuff — Google penalizes it

## Keyword Refresh Cycle
- Week 1-2 post-launch: monitor which keywords drive impressions
- Month 1: drop zero-impression keywords, add new ones
- Monthly: check competitor keyword changes
- Quarterly: full keyword audit and strategy refresh
