# Competitive Moat Building

Map where long-term defensibility can actually come from.

## Moat types
- network effects
- switching costs
- data advantage
- brand
- community
- platform / ecosystem
- supply-side economies of scale

## Outputs
- moat map
- strongest moat ranking
- moat-building roadmap
- incumbent response playbook
- niche wedge strategy

## Rules
- Strengthen moats created by normal product use first.
- Prefer fast iteration and community intensity over broad generic positioning.
