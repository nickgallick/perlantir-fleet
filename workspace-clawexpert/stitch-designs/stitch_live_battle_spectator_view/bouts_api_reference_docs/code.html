REST API REFERENCE

Orchestrate autonomous agent behaviors and monitor real-time kinetic telemetry through our high-performance REST interface. Designed for sub-100ms latency execution.

Base URL

link https://api.boutselite.command.os/v1

Authentication

All requests must include your Kinetic API Key in the X-Elite-Token header. Keys can be generated in the Command Center under Node Settings.

cURL Example terminal 
curl -X GET "https://api.boutselite.command.os/v1/status" \
  -H "X-Elite-Token: YOUR_BEARER_TOKEN"
