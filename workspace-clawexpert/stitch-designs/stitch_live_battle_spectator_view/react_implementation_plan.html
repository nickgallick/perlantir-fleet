# Bouts Elite: Full React Implementation Strategy ## 1. Tech Stack Recommendation * **Framework:** Next.js (App Router) for SEO-friendly landing pages and optimized dashboard performance. * **Styling:** Tailwind CSS (already integrated in the HTML designs). * **Icons:** Lucide-React or Heroicons. * **State Management:** TanStack Query (React Query) for server state (leaderboards, agent status) and Zustand for light client-side state (UI toggles). * **Animation:** Framer Motion for the "AI kinetic" transitions. ## 2. Component Architecture ### Shared Components (`/components/shared`) * `Navigation.tsx`: The floating pill nav for public pages and the sidebar for authenticated views. * `Footer.tsx`: The multi-column technical footer. * `StatCard.tsx`: The high-density cards used across the dashboard and home. * `WeightClassBadge.tsx`: Reusable badges for Lightweight, Contender, etc. ### Core Layouts (`/app/layouts`) * `PublicLayout.tsx`: Includes the floating pill nav and main footer. * `DashboardLayout.tsx`: Includes the sidebar, user header, and minimized footer. ## 3.
