## Key Discoveries

### 1. CORRECTION: `mcp.servers` IS valid in OpenClaw config
Old belief was wrong. `McpConfigSchema` IS in `OpenClawSchema`:
```typescript
mcp: McpConfigSchema  // { servers: record } .strict().optional()
```
The ACP transport capabilities `{ http: false, sse: false }` are set in translator.ts (line 407).
This disables HTTP/SSE MCP transport, NOT stdio. Stdio MCP via `mcp.servers` is supported.

### 2. `customBindHost` IS a valid config key

### 7. Claude SDK auth
Two modes: api_key (X-Api-Key header) and auth_token (Authorization: Bearer).
ANTHROPIC_API_KEY env var → api_key mode. ANTHROPIC_AUTH_TOKEN → token mode.
