# SKILL 77: Trademark & Brand Clearance

## Purpose
Protect your brand before public launch. A trademark conflict discovered after you've built user recognition is expensive to fix. Search first, file early, monitor always.

## Pre-Launch Brand Clearance Process

### Step 1 — Comprehensive Name Search
**USPTO TESS** (tess2.uspto.gov): search for existing federal trademark registrations
- Search: exact match, similar spellings, phonetic equivalents, similar meanings in English and other languages
- Search by international class (not just name)

**Common Law Search**
- Google: "[brand name]" — find businesses using the name even without registered trademarks
- State trademark databases: each state maintains a trademark registry
- Domain registrars: who owns [brand].com, [brand].io, [brand].ai?
- Social media: Twitter/X, Instagram, TikTok, Discord, Reddit — is the handle taken?
- App stores: Apple App Store, Google Play — similar app names?

### Step 2 — International Classes for Your Products

| Class | Covers | Nick's Products |
|-------|--------|----------------|
| Class 9 | Software, apps, downloadable content, AI systems | All products |
| Class 36 | Financial services, insurance, prediction markets | Prediction market |
| Class 41 | Entertainment, gaming, competitions, education | Agent Sparta, competitions |
| Class 42 | SaaS, computer programming, technology consulting | All products |

**Agent Sparta minimum**: file in Classes 9, 41, and 42
**Prediction Market**: add Class 36

### Step 3 — Assess Conflicts

| Scenario | Risk Level | Action |
|---------|-----------|--------|
| Identical mark in same class | 🔴 HIGH | Choose a different name |
| Similar mark in same class | 🟡 MEDIUM | Get trademark attorney opinion; may need to modify |
| Same mark in different class | 🟢 LOW | Usually fine; evaluate consumer confusion |
| Common law use (no registration) | 🟡 MEDIUM | Prior use creates rights; may force you to change |

### Step 4 — File USPTO Application
- **TEAS Plus**: $250 per class (requires selecting from ID Manual for goods/services description)
- **TEAS Standard**: $350 per class (allows custom description)
- **Use basis**: "intent to use" (1(b)) if not yet using the mark in commerce
- **Use in commerce**: (1(a)) if already using; requires specimen (screenshot, app store listing)
- **Timeline**: 8–12 months from filing to registration if no opposition
- **Renewal**: renew between years 5–6 (Section 8 declaration), then every 10 years

### Step 5 — Monitor After Filing
- USPTO Official Gazette: published weekly; published applications pending 30-day opposition window
- Monitoring services: Corsearch, TrademarkNow, Dennemeyer — automated conflict detection
- Google Alerts: "[brand name] trademark" and "[brand name]" for new uses

## Domain Name Strategy
| Domain | Priority | Action |
|--------|---------|--------|
| [brand].com | ⚠️ Critical | Register immediately |
| [brand].ai | High | Register immediately |
| [brand].io | High | Register immediately |
| [brand].xyz | Medium | Register proactively |
| [brand].net | Medium | Register proactively |
| Common misspellings | Low | Register defensively |

**If .com is taken**:
- Evaluate acquisition via domain brokers (Sedo, Afternic, GoDaddy broker service)
- Typical cost: $1K–$100K+ depending on the domain
- Alternative: use [brand].ai as primary if the .com owner is unresponsive and the price is unreasonable

**Domain privacy**: use WHOIS privacy (available from most registrars) to hide personal contact information.

**UDRP** (if cybersquatted): if someone registered your trademark as a domain in bad faith, file a Uniform Domain-Name Dispute-Resolution Policy complaint with WIPO. Cost: ~$1,500 for single-panelist case. Success rate: high if you have a registered trademark and clear bad faith.

## Social Media Handle Strategy
Secure on ALL platforms before ANY public announcement:
- Twitter/X, Instagram, TikTok, Discord (server), Telegram (channel), Reddit (subreddit), YouTube, LinkedIn (company page), GitHub (organization), Facebook (page)

**If a handle is taken**:
- Most platforms have trademark-based dispute processes after you have a registered trademark
- Without a registered trademark: very limited recourse

## Enforcement
- If someone uses a confusingly similar mark after your registration: cease and desist letter first (from outside counsel)
- If they don't stop: trademark infringement lawsuit in federal court
- Damages: actual damages + infringer's profits + attorney's fees (for willful infringement)
- Registered trademark is required for maximum damages — another reason to file early
