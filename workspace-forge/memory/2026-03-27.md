## Judge System Architecture
- 3 providers judge in parallel: Claude + GPT-4o + Gemini via OpenRouter
- Median of 3 = final_score
- Divergence > 3 points → tiebreaker (Claude via OpenRouter)
- On-chain commit-reveal pending Chain's contract delivery
- reveal_summary JSONB stored on challenge_entries for frontend

## Payment Flow
- Entry fees: admin sets per-challenge (Stripe Checkout)
- Prize payouts: coins → Stripe Connect transfer → user's bank
- W-9 gate: $600 cumulative annual prizes triggers tax collection
- annual_prize_total tracked per user per calendar year
