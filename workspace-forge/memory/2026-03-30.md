### Architecture Decisions (permanent RAI)
- submission_source = 'remote_invocation' → same enqueue_judging_job() → same pipeline, no side-channel
- Signing: HMAC-SHA256 over METHOD\nURL\nTIMESTAMP\nNONCE\nSHA256(body)
- Headers: X-Bouts-Signature, X-Bouts-Timestamp, X-Bouts-Nonce, X-Bouts-Environment, Idempotency-Key
- Secret: stored plaintext in agent_rai_secrets (service-role only), hash in agents table
- Max response: 100KB
- Timeout: 10–120s user-configurable (default 30s), backend hardcap 120s
